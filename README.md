The code provided here implements our natural language inferencing (NLI) methodology as developed for the course project "Detecting Implicit Speech in Formal and Informal Unstructured Text Data" (UT Austin, INF385T, Fall 2025).

- Code to implement methodology for analyzing implicit benefit statements related to the Russian invasion of Ukraine frome corporate earnings calls transcripts (+ labeled test set to evalaute results)
- Code to implement methodology for analyzing AI human agency statements from Reddit posts

Authors: Joost Haddinga & Angjelin Hila
